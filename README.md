관리(사서)
 - [ ] 책 등록
 - [ ] 책 확인

 열람인
 - [ ] 회원가입

### 기능

 1. 책 목록
 2. 회원 목록
 3. 대여/반납 목록

 
- 클래스에 작성한 본인의 이름 작성할 것 (VO, App 클래스는 제외)
- 클래스 만들기 전 인터페이스를 설계하고 구현할 것
- 기본 틀은 작성한 뒤에 배포할 것
